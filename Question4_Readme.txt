To run this program, open it in a C++ compiler or
go to https://onlinegdb.com open this the Question3.cpp
and run the code. 
Follow the instructions:
Enter the first string and press enter
Enter the second string and press enter
In the end, enter the number of interactions and press enter. 
After that, the program will display the result in the console. 

Note: there is no verification for String1 or string2 but they should have less than 100 characters. 
Note: if you enter with a number of interactions over 100, the program will do nothing 
and will display an error message. 

The results will be displayed in the console.
